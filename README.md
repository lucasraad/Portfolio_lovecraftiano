# O Grimório Digital - Portfólio Lovecraftiano

Um portfólio web único com temática Lovecraftiana e Cthulhu, especialmente desenvolvido para programadores Python que desejam se destacar com uma apresentação memorável e atmosférica.

## 🐙 Visão Geral

Este portfólio combina a atmosfera sombria e misteriosa dos contos de H.P. Lovecraft com elementos modernos de web design, criando uma experiência imersiva que reflete a profundidade e complexidade do desenvolvimento Python.

### Características Principais

- **Temática Lovecraftiana Autêntica**: Terminologia, símbolos e atmosfera inspirados nos Mitos de Cthulhu
- **Design Responsivo**: Funciona perfeitamente em desktop, tablet e mobile
- **Animações Suaves**: Efeitos visuais que criam uma experiência envolvente
- **Performance Otimizada**: Código limpo e otimizado para carregamento rápido
- **Acessibilidade**: Suporte para leitores de tela e navegação por teclado

## 🎨 Paleta de Cores Temática

- **Preto Abissal**: `#0a0a0a` - Cor de fundo principal
- **Verde Fosforescente**: `#39ff14` - Acentos e elementos interativos
- **Verde Eldritch**: `#1a4a3a` - Elementos secundários e bordas
- **Dourado Ancestral**: `#d4af37` - Destaques especiais
- **Branco Osso**: `#e8e8dc` - Texto principal
- **Vermelho Sangue**: `#8b0000` - Elementos de alerta

## 🛠️ Personalização

### Conteúdo
1. **Informações Pessoais**: Edite as seções de biografia e contato
2. **Projetos**: Substitua os projetos exemplo pelos seus próprios
3. **Experiência**: Atualize a timeline com sua trajetória profissional
4. **Skills**: Ajuste as habilidades e níveis conforme sua expertise

### Visual
1. **Cores**: Modifique as variáveis CSS no início do `main.css`
2. **Fontes**: Altere as famílias de fonte nas variáveis CSS
3. **Imagens**: Substitua os assets na pasta `assets/`
4. **Animações**: Ajuste velocidades e efeitos no `animations.css`

### Funcionalidades
1. **Formulário**: Configure o endpoint de envio no `main.js`
2. **Analytics**: Adicione seu código de tracking
3. **SEO**: Atualize meta tags e structured data
